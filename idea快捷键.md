# idea快捷键

**Ctrl + 鼠标左击**
①查看方法或者类的源码，若为类的实例化的对象，则跳转到该对象的实例化处
②查看类的对象实例的使用位置
**Ctrl + Alt + T**
surround with…用if,while,try…catch…等的语句块将一段代码包起来
**Ctrl + P**
查看类的构造器所需参数
**Ctrl + F12**
就地查找 字段 方法名
**Ctrl + H**
查看继承的层级关系
**Ctrl + 7**
查看类里的各个结构
**Ctrl + F**
在当前页面进行查找
**Ctrl + R**
在当前页面进行替换
**Ctrl + Alt + L**
快速格式化
**Ctrl + O**
重写方法
**Ctrl + Alt + left****
返回上次光标所在位置
**Ctrl + Alt + right**
返回下次光标所在位置，即与Ctrl + Alt + left进行来回切换
**Ctrl + Alt + 鼠标左击**
查看方法所属的实现类
**Shift + F6**
同时改变多处自定义的名称

查看类图 快捷键 ： **Ctrl + shift + Alt +U**   Ctrl + Alt + B 添加类图关系 Ctrl + A 添加所有类图
